# 05-BugFix
Sistema de Administración de relaciones con clientes

Descripción del Proyecto: Plataforma CRM para Perfumería
Este proyecto consiste en el desarrollo de una plataforma CRM (Customer Relationship Management) diseñada específicamente para una perfumería. El objetivo principal es optimizar la gestión de clientes, fortalecer la relación con ellos y mejorar la experiencia de compra mediante un sistema digital centralizado y fácil de usar.
La plataforma permite registrar y organizar información clave de los clientes, como datos personales, historial de compras, preferencias aromáticas, fechas especiales (cumpleaños, aniversarios), y programas de fidelización. A través de estas funciones, la perfumería puede ofrecer recomendaciones personalizadas, promociones exclusivas y recordatorios automáticos, fomentando la lealtad y aumentando las ventas.
